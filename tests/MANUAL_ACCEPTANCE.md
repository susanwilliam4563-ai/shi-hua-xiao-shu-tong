# 可重复手工验收

1. 清空站点数据，在 375px 宽度打开首页；三步内填写昵称、年级、目标，进入今日闯关。
2. 打开任一任务，依次完成猜诗、语音朗读、画面理解、点击排序、关键词提示、录音/自评和反馈；刷新后在诗词地图看到掌握状态。
3. 返回今日闯关，确认对应任务已打勾；关闭再打开页面，完成状态仍在。
4. 在游戏馆依次玩看图猜诗、上下句、错字侦探、季节分类；家长中心的 7 天柱状图与掌握分布发生变化。
5. 在诗词地图分别选择年级、上下册、状态，确认卡片筛选正确。
6. 拒绝麦克风权限，确认仍显示自评按钮；在不支持语音合成的浏览器中仍可继续下一步。
7. 用键盘 Tab/Enter 完成主要按钮操作；在 375px、768px、1440px 检查无横向溢出和遮挡。
