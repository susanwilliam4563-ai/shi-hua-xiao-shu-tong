# 诗画小书童实施计划

## 范围与验收

独立、零外部服务的响应式 Web MVP。儿童档案、学习记录、任务、设置均存入 localStorage；语音使用浏览器合成，录音使用 MediaRecorder 并提供自评降级。

## 页面与组件

- 首页、三步引导、今日闯关、诗词地图、七步学习、游戏馆、诗词树、家长中心、设置。
- 通用组件：顶部导航、进度条、诗词卡、状态徽标、反馈提示、确认对话框。
- 学习组件：猜诗、听读、画面理解、句序拼图、关键词消失、录音背诵、完成反馈。

## 数据模型

- `Poem`：教材信息、逐句原文/拼音/解释、意象、主题、题目与媒体占位字段。
- `PoemProgress`：掌握等级、能力分、错误句、提示数、复习间隔与学习记录。
- `AppState`：儿童档案、设置、每日活动、诗词进度与成就。
- `storage` 适配层隔离 localStorage，后续可替换为 Supabase。

运行 `npm run lint && npm run typecheck && npm test && npm run build`；再按 `tests/MANUAL_ACCEPTANCE.md` 在 375px、平板和桌面完成主流程。
