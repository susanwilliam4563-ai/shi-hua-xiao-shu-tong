# 诗画小书童

面向小学一至三年级儿童的古诗词理解、背诵与长期记忆 Web MVP。每天用约 10 分钟完成“看图理解 → 听读 → 主动回忆 → 背诵 → 间隔复习 → 易错强化”。

## 功能

包含游客引导、每日 3 个自适应任务、按年级与诗名自由选诗、25 首诗词与六个主题地图、统一八关学习流程、单诗专项复习与专属游戏、十四种轮换游戏和综合闯关、诗词卡与画卷奖励、自动复习、孩子创作和家长学习报告。每首诗都配有本地原创中国风插图、朗读、儿童诗意讲解、重点词语、排序、十轮多形式填空和背诵挑战。进度刷新或退出后保留。

## 技术与结构

浏览器原生 ESM、HTML5、CSS 与 localStorage；Node 内置测试和零依赖构建。`src/poems.js` 是内容层，`src/review.js` 是可测试领域逻辑，`src/storage.js` 是可替换的数据访问层，`src/app.js` 负责路由与交互。

## 本地启动

需要 Node.js 18 或更高版本。

```bash
cd poem-scholar
npm run dev
```

浏览器打开 `http://localhost:4173`。

## 检查、测试与构建

```bash
npm run lint
npm run typecheck
npm test
npm run build
```

生产文件输出到 `dist/`，可部署到任意静态托管服务。公开访问时，每位访客的数据仍只保存在自己的浏览器中，互不共享。

## 独立公开部署

仓库根目录的 `render.yaml` 是 Render Blueprint，使用 `pnpm build` 生成纯静态站点并发布 `dist/`。在 Render 连接本仓库后，每次推送到主分支会自动部署；生成的 `onrender.com` 地址可由普通浏览器直接访问，不依赖 ChatGPT Sites，也不要求访客登录。

离线功能保留：访客首次联网打开后，核心页面和全部诗词插图会缓存在自己的设备中，之后可在无网络时继续使用。

## 数据与隐私

本地 MVP 只保存昵称、年级、设置和学习记录到当前浏览器的 localStorage。录音仅在当前页面内临时回放，不上传、不持久化；无广告追踪、社交和公开排行。

正式公开运营、账号系统、收费、支付、云端录音或内容后台上线前，必须另行完成儿童个人信息保护、监护人同意、内容版权、音频与图片授权、网站备案和运营合规审查。

## 已知限制

- 语音朗读质量取决于浏览器中文语音；尚未接入真人音频、云端识别与内容后台。
- 诗词插图为项目内置原创画面；后续替换或新增图片时仍需登记来源并核对诗句意象。
- localStorage 不跨设备同步，清除浏览器数据会丢失进度。
- 当前收录为统编版一至三年级诗词精选；不同学年教材版次如有调整，应以孩子手中教材为准。
- 浏览器端自动化以可重复手工验收清单替代；关键复习逻辑已有自动单元测试。
