# 诗画小书童

面向小学一至三年级儿童的古诗词理解、背诵与长期记忆 Web MVP。每天用约 10 分钟完成“看图理解 → 听读 → 主动回忆 → 背诵 → 间隔复习 → 易错强化”。

## 功能

包含三步引导、每日 3—5 个自适应任务、12 首诗词地图、统一七步学习流程、八种专项游戏和综合闯关、诗叶积分、诗词树与徽章、基于实际记录的家长中心，以及声音、拼音、字体和时长设置。每局游戏为 8 题、约 5—10 分钟，包含难度递进、错题回顾和结算。进度刷新后保留。

## 技术与结构

浏览器原生 ESM、HTML5、CSS 与 localStorage；Node 内置测试和零依赖构建。`src/poems.js` 是内容层，`src/review.js` 是可测试领域逻辑，`src/storage.js` 是可替换的数据访问层，`src/app.js` 负责路由与交互。

## 本地启动

需要 Node.js 18 或更高版本。

```bash
cd poem-scholar
npm run dev
```

浏览器打开 `http://localhost:4173`。

## 检查、测试与构建

```bash
npm run lint
npm run typecheck
npm test
npm run build
```

生产文件输出到 `dist/`，可部署到任意静态托管服务。公开访问时，每位访客的数据仍只保存在自己的浏览器中，互不共享。

## 数据与隐私

本地 MVP 只保存昵称、年级、设置和学习记录到当前浏览器的 localStorage。录音仅在当前页面内临时回放，不上传、不持久化；无广告追踪、社交和公开排行。

正式公开运营、账号系统、收费、支付、云端录音或内容后台上线前，必须另行完成儿童个人信息保护、监护人同意、内容版权、音频与图片授权、网站备案和运营合规审查。

## 已知限制

- 语音朗读质量取决于浏览器中文语音；尚未接入真人音频、云端识别与内容后台。
- 场景图使用轻量本地占位表达，正式版应替换为已授权插画。
- localStorage 不跨设备同步，清除浏览器数据会丢失进度。
- 教材归属为演示配置，正式上线前需按目标教材年份逐首复核。
- 浏览器端自动化以可重复手工验收清单替代；关键复习逻辑已有自动单元测试。
